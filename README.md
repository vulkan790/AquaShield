## О проекте

**AquaShield** - это автономная система обнаружения протечек воды с автоматическим перекрытием. 

### Ключевые особенности

Состоит из трёх устройств: **сенсора**, **хаба** и **реле**.
**Сенсор** - часть системы, цель которой оповещать протечку с помощью звука (зуммера). Устанавливается в зонах риска.
**Реле** - часть системы, цель которой перекрывать подачу воды в случае протечки. Подключается к водопроводу.
**Хаб** - часть системы, цель которой связывать сенсор и реле. 

Пользователь может настроить самостоятельно как оповещать протечку воды: либо просто с помощью сенсора, либо сразу перекрывать.
Весь интерфейс системы реализуется в Telegram и VK ботах.
Если обнаруживается протечка воды, сенсор передает сигнал о протечке по радиоканалу 433 Мгц и оправляет уведомление в Telegram и VK.
