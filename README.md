## О проекте

**AquaShield** — автономная система обнаружения протечек воды с возможностью автоматического перекрытия.

### Ключевые особенности

Система состоит из трёх устройств:

- **Сенсор** — обнаруживает воду (зуммер + радиосигнал). Устанавливается в зонах риска.
- **Хаб** — принимает сигнал от сенсора, управляет реле и отправляет уведомления.
- **Реле** — перекрывает подачу воды при срабатывании системы.

Пользователь может настроить сценарий:
- Только оповещение (звук + сообщение в бот)
- Автоматическое перекрытие воды
- Удалённое управление через Telegram / VK

Если обнаруживается протечка, сенсор передаёт сигнал по радиоканалу 433 МГц, хаб активирует реле, и боты отправляют уведомление.
